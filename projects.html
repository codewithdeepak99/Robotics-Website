Our Innovation Lab

Exploring the frontiers of automation and robotics.

Self Balancing Bot

A self-balancing robot works on the principle of an inverted pendulum. It continuously measures tilt using an MPU6050 sensor and applies real-time motor correction to maintain balance.

Mark 1.6

Mark 1.6 is a Bluetooth-controlled robotic arm mounted on a 4WD chassis. It uses a 3D-printed arm and NRF modules for wireless communication, making it flexible, lightweight, and cost-effective.

Line Follower Bot

The line follower robot tracks a predefined path using IR sensors. Real-time sensor feedback enables smooth and accurate navigation, commonly used in automation and robotics competitions.

Pluto X

Pluto X is an open-source nano drone developed at IIT Bombay. It supports Wi-Fi camera streaming, ROS integration, and autonomous flight experiments.

Hexapod

Hexapod robots provide stable movement across uneven terrain. Built using Arduino Nano and servo motors, it enables multi-degree freedom and gait experimentation.

3D Printing

The creation of 3D printed objects using additive processes. We utilize this to create custom chassis, gears, and structural components for our robots.

Automated Bot

An autonomous bot that navigates walls using sharp laser sensors. It implements a closed-loop feedback mechanism to correct alignment and count laps/objects.

Radar Navigation

A sophisticated navigation system utilizing radar technology for obstacle detection, mapping, and path planning in complex environments.

Drone

Custom-built aerial vehicles designed for surveillance, aerial photography, and stability testing under various atmospheric conditions.
